# README

基于stm32 standard peripheral libraries的USART串口收发字符练习

## 核心功能

* 功能点 1：实现了DevEBox 基础板stm32f407zgt6的PA9 PA10转换TX1 RX1的串口发送接收功能
* 功能点 2：打印输入字符的ASCII码
* 功能点 3：使用前向声明实现数据封装，降低模块间的编译依赖，提升编译速度

## 运行环境与依赖

* **操作系统**：Windows 11
* **依赖库**：stm32 standard peripheral libraries
* **硬件**：USB转串口 CH340
